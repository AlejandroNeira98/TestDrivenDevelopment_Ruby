# Test Driven Development in Ruby (Solver)

> A Simple Ruby Based Solver create using TDD technique


## Built With

- Ruby

## Getting Started

To get a local copy follow these simple example steps.  

1.- Open the terminal window and clone the repository using this command:  
`git clone git@github.com:AlejandroNeira98/TestDrivenDevelopment_Ruby.git` 

2.- Change the directory to the school-library directory  
`cd TestDrivenDevelopment_Ruby`  
3.- Install the project's dependencies by running this command:   
`bundle install`
